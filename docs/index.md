# 学習コンテンツ

エンジニアのための実践的な学習資料を提供します。

## このサイトについて

実務で役立つエンジニアリングプラクティスを体系的に学べる学習コンテンツを提供しています。

## コンテンツ

### モニタリング・アラート

システムの安定運用に欠かせないモニタリングとアラートの設計について学べます。

- **アラート設計**: ユーザー視点でのアラート分類とレベル設定
- **多層モニタリング**: システムを多層的に監視する重要性と実践方法
- **クラウド環境**: クラウド特有のリソース監視とアラートの考え方

[モニタリング・アラートを学ぶ →](monitoring/index.md){ .md-button .md-button--primary }

### Infrastructure as Code (IaC)

インフラストラクチャをコードで定義・管理する手法について学べます。

- **基礎知識**: IaCのメリット、宣言的定義と命令的定義の違い
- **実践スキル**: コーディング原則、テスト、ディレクトリ構成、管理単位のベストプラクティス
- **参考リンク**: AWS CDK・Terraformの公式サンプルやベストプラクティス集

[IaCを学ぶ →](iac/index.md){ .md-button .md-button--primary }

### セキュリティ対策

システムを安全に運用するためのセキュリティ対策と運用プラクティスについて学べます。

- **基本原則**: 多層防御、最小権限、ゼロトラスト、リスクベースのアプローチ
- **予防的対策**: 認証・認可、暗号化、ネットワークセキュリティ
- **検知と対応**: ログ監視、インシデント対応
- **セキュリティ運用**: 脆弱性管理と重大度評価
- **開発プロセス**: セキュアコーディング、SAST/DAST

[セキュリティ対策を学ぶ →](security/index.md){ .md-button .md-button--primary }

ソフトウェア開発を成功に導くためのチーム開発プラクティスについて学べます。

- **チームの土台**: 価値観の共有、期待値設定、コミュニケーションの取り方
