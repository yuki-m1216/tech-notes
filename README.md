# 学習コンテンツ

実務で役立つ技術知識を体系的に学べる学習コンテンツサイトです。

🔗 **サイトURL**: https://yuki-m1216.github.io/tech-notes/

## 📚 コンテンツ

### モニタリング・アラート

システムの安定運用に欠かせないモニタリングとアラートの設計について学べます。

- **アラート設計**: ユーザー視点の分類、アラートレベルの決め方
- **多層モニタリング**: 概念と実践例
- **クラウド環境での考え方**: クラウド特有の設計、リソース監視
- **実践編**: よくある失敗例

### Infrastructure as Code (IaC)

インフラストラクチャをコードで定義・管理する手法について学べます。

- **基礎知識**: IaCのメリット、宣言的定義と命令的定義の違い
- **実践スキル**: コーディング原則、テスト、ディレクトリ構成、管理単位のベストプラクティス
- **参考リンク**: AWS CDK・Terraformの公式サンプルやベストプラクティス集

### セキュリティ対策

システムを安全に運用するためのセキュリティ対策と運用プラクティスについて学べます。

- **基本原則**: 多層防御、最小権限、ゼロトラスト、リスクベースのアプローチ
- **予防的対策**: 認証・認可、暗号化、ネットワークセキュリティ
- **検知と対応**: ログ監視、インシデント対応
- **継続的なセキュリティ**: 脆弱性管理、セキュア開発、コンプライアンス対応

## 🚀 ローカル開発

### 必要な環境

- Python 3.8以上
- pip

### セットアップと起動

```bash
# 依存関係のインストール
pip install mkdocs-material

# ローカルサーバーの起動
mkdocs serve
```

ブラウザで http://127.0.0.1:8000/ にアクセスしてください。

### ビルド

```bash
mkdocs build
```
